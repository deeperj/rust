Using the Set type

Category: Data Types
Description: Create a set of particular unicode characters using a Microsoft.FSharp.Collections.Set.
Code: 
let SampleSet1() =
    let data = "The quick brown fox jumps over the lazy dog" 
    let set = 
        data.ToCharArray()
        |> Set.ofSeq
    for c in set do 
        printfn "Char : '%c'" c
